Delorean installation for Ubuntu 12.10 (works on my 13.04)

`sudo add-apt-repository ppa:killhellokitty/themes.ppa` <br />
`sudo apt-get update` <br />
`sudo apt-get install delorean-dark-theme-3.6` <br />
